Pods use hostNetwork, so each agent binds directly on the node IP.
The ServiceAccount is granted the hostnetwork-v2 SCC via RBAC.

To list all agents:
  kubectl get pods -l app.kubernetes.io/name=zabbix-stress-agent -n {{ .Release.Namespace }}
